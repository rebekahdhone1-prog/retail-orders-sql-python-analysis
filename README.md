# Retail Orders Sales Analysis (SQL Server + Python)

## Overview
End-to-end retail sales analysis project using SQL Server and Python
to extract insights related to sales, revenue, and growth.

## Tools & Technologies
- SQL Server
- SQL
- Python
- Pandas
- SQLAlchemy
- Kaggle
- Jupyter Notebook

## Workflow
- Downloaded retail orders dataset using Kaggle API
- Cleaned and transformed data using Pandas
- Loaded processed data into SQL Server using SQLAlchemy
- Performed analytical SQL queries in SQL Server
- Combined Python and SQL analysis for insights

## Analysis Performed
- Top-selling products
- Region-wise sales performance
- Month-over-month growth
- Category and sub-category trends
- High-growth product identification

## Key Learnings
- End-to-end data analysis pipeline
- SQL Server and Python integration
- Writing analytical SQL queries
- Business-focused data insights
